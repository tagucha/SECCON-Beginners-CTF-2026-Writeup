# twins

これは、RSA暗号がテーマになった問題です。

```python
n1 = p * q1
n2 = p * q2
```

のように、RSA暗号の安全性の源である`n`の素因数分解がユークリッドの互助法により容易にできる構造となっています。

```python
p = gcd(n1, n2)
q1 = n1 // p
```

というふうに素因数を求めることができるので、あとは通常のRSA暗号の復号を行えば終わりです。

```python
phi = (p - 1) * (q1 - 1)
d = pow(e, -1, phi)
m = pow(c, d, n1)
```
